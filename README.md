eswiftBLE
==================
## Introduction
基于苹果官方外设库CoreBluetooth的一款Swift语言的IOS Demo程序。经测试已实现
外设BLE的查找，连接，服务发现，数据读写，通知开关和读取。程序提供了较好的中文注释，可以使开发者理解和快速应用蓝牙外设。  

欢迎提交建议和意见，若觉得有帮助也十分欢迎star和fork。

开发语言：swift 3.0  
平台:Xcode 7.1

## Package:
BTM: CoreBluetooth
